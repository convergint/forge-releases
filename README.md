# Homebrew Forge

Homebrew tap for [Forge](https://github.com/convergint/forge) ‚ÄĒ an AI-first quality pipeline for your repo.

## Install

```bash
brew tap convergint/forge
brew install convergint/forge/forge
```

## Upgrade

```bash
brew upgrade convergint/forge/forge
```

## Windows

Download the `.zip` for your architecture from the [latest release](https://github.com/convergint/homebrew-forge/releases) and add `forge.exe` to your PATH.
